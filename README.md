# ML and DL Algorithms — Short Notes

---

# 1. Linear Regression
- Supervised learning algorithm.
- Used to predict continuous values.
- Finds a straight-line relationship between input and output.
- Example: House price prediction.


---

# 2. Polynomial Regression
- Extension of linear regression.
- Models curved relationships using polynomial terms.
- Useful when data is non-linear.

---

# 3. Ridge Regression
- Linear regression with L2 regularization.
- Reduces overfitting by shrinking coefficients.
- Handles multicollinearity.

---

# 4. Lasso Regression
- Linear regression with L1 regularization.
- Reduces overfitting and removes less important features.
- Useful for feature selection.

---

# 5. Logistic Regression
- Classification algorithm.
- Predicts probability between 0 and 1.
- Mainly used for binary classification.

---

# 6. Naïve Bayes
- Probability-based classification algorithm.
- Uses Bayes theorem with independence assumption.
- Fast and efficient for text classification.

---

# 7. Decision Tree
- Tree-based supervised algorithm.
- Splits data using conditions.
- Easy to understand and visualize.


# 8. Random Forest
- Collection of multiple decision trees.
- Combines outputs for better accuracy.
- Reduces overfitting.

---

# 9. AdaBoost
- Boosting algorithm.
- Combines weak learners into a strong learner.
- Focuses more on wrongly classified samples.

---

# 10. Support Vector Machine (SVM)
- Classification and regression algorithm.
- Finds optimal hyperplane between classes.
- Effective for high-dimensional data.

---

# 11. K-Nearest Neighbor (KNN)
- Instance-based learning algorithm.
- Classifies data using nearest neighbors.
- Simple and easy to implement.

---

# 12. K-Means Clustering
- Unsupervised clustering algorithm.
- Groups similar data into K clusters.
- Uses centroid-based clustering.

---

# 13. Principal Component Analysis (PCA)
- Dimensionality reduction technique.
- Converts large features into smaller important components.
- Improves speed and visualization.

---

# 14. Apriori Algorithm
- Association rule learning algorithm.
- Finds frequent itemsets in datasets.
- Used in market basket analysis.

---

# 15. Q-Learning
- Reinforcement learning algorithm.
- Learns using rewards and penalties.
- Agent improves through trial and error.

---

# 16. Deep Learning
- Subset of machine learning using deep neural networks.
- Learns complex patterns from large data.

---

## 16.1 Deep Neural Network (DNN)
- Neural network with multiple hidden layers.
- Used for classification and prediction tasks.

---

## 16.2 Convolutional Neural Network (CNN)
- Specialized for image processing.
- Uses convolution and pooling layers.
- Detects patterns, edges, and objects.


---

## 16.3 Transformer
- Deep learning architecture using attention mechanism.
- Processes sequence data efficiently.
- Used in NLP and generative AI.
